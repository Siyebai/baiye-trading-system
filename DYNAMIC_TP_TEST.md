# 动态TP测试报告
**结论**: 动态TP（ADX>=30时放大，ADX<20时缩小）总体**不改善**，不建议采用

| 品种 | 静态TP月均 | 动态TP月均 | 判断 |
|------|-----------|-----------|------|
| BTC | +1.4% | -2.3% | ❌变差 |
| ETH | +1.0% | +0.9% | ≈持平 |
| SOL | -1.1% | -0.5% | ✅轻微改善 |
| BNB | +2.6% | +2.4% | ≈持平 |
| LINK | +5.3% | +3.3% | ❌变差 |
| POL | -2.2% | -1.8% | ≈持平 |

**原因分析**: 动态TP降低WR（放大TP时命中率下降），对于本策略的均值回归逻辑反而有害。
**保持**: 固定TP倍数（策略核心参数），不引入动态TP。
